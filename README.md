# 💼 Sitio Web - Romina Sánchez Abogada

Sitio web profesional desarrollado para una abogada, enfocado en presencia online, contacto con clientes y posicionamiento en buscadores (SEO).

---

## 🚀 Demo

🔗 https://romina-sanchez-abogada.vercel.app/

---

## 🧠 Descripción

Este proyecto es una **One Page moderna y responsive** que presenta:

* Información profesional
* Servicios legales
* Sección de contacto
* Navegación con scroll suave
* Modo claro / oscuro

El objetivo principal es brindar una **imagen profesional y facilitar la captación de clientes**.

---

## 🛠️ Tecnologías utilizadas

* ⚛️ React
* ⚡ Vite
* 🎨 CSS (custom properties + responsive design)
* 🌐 Vercel (deploy)

---

## ✨ Funcionalidades

* 🔄 Scroll automático entre secciones
* 📍 Detección de sección activa (IntersectionObserver)
* 🌙 Modo oscuro / claro
* 📱 Diseño responsive (mobile-first)
* 🎯 Optimización SEO básica

## ⚙️ Instalación

Clonar el repositorio:

```bash
git clone https://github.com/tu-usuario/tu-repo.git
```

Instalar dependencias:

```bash
npm install
```

Ejecutar en desarrollo:

```bash
npm run dev
```
